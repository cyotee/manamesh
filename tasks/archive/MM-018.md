## Technical Details

### Source

[OpenGameArt cards_0.zip](https://opengameart.org/sites/default/files/cards_0.zip)

### Manifest Structure

```json
{
  "manifest_version": "1.0",
  "pack_id": "standard-playing-cards",
  "pack_type": "card_faces",
  "name": "Standard 52-Card Deck",
  "game_module": ["war", "poker"],
  "version": "1.0.0",

  "asset_schema": {
    "id_format": "suit-rank",
    "asset_types": ["card_face", "card_back"]
  },

  "assets": [
    { "id": "hearts-A", "type": "card_face", "file": "cards/hearts_A.png" },
    { "id": "hearts-2", "type": "card_face", "file": "cards/hearts_2.png" },
    // ... all 52 cards
    { "id": "back", "type": "card_back", "file": "cards/back.png" }
  ]
}
```
